Code for Ichinomiya

はじめに

皆さんはシビックテック（CivicTech）をご存知ですか？

シビックテックとは、市民（Civic）が自らデジタル技術（Tech）を用いて、地域課題を解決していく取り組みのことです。

Code for Ichinomiyaは、愛知県一宮市にて、シビックテックの活動を行なっています。
