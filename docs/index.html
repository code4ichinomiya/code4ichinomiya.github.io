はじめに

皆さんはシビックテック（CivicTech）をご存知ですか？市民（Civic）が自らデジタル技術（Tech）を用いて、地域課題（Region）を解決していく取り組みのことです。

日本では、東日本大震災後、Code for Japanという団体が誕生したことがきっかけで、シビックテックの活動が増えています。

Code for Ichinomiyaは、愛知県一宮市にて、シビックテックの活動を行なっています。
